##crowdring
==============

+ crowdring, from https://github.com/mbelinsky/Crowdring

+ Pre Install
  
  + Install Redis
  
      + download redit from [Here](http://redis.io/)

      + run `tar xvf redis.version.tar`

      + `cd redis.version/`

      + run `make install`
  
  + Install Ruby 1.9.3

+ To install. 

  + `git clone git@github.com:nherzing/crowdring.git`

  + `cd crowdring`

  + `bundle install`
